# 3D Endless Runner Game

A Temple Run inspired 3D endless runner developed using Unity and C#.

## Features
- Lane switching mechanics
- Procedural obstacle spawning
- Infinite level generation
- Coin collection system
- Score tracking
- Game Over system
- Difficulty progression

## Technologies Used
- Unity
- C#
- Visual Studio

## Controls
- A / Left Arrow → Move Left
- D / Right Arrow → Move Right
- Space → Jump
